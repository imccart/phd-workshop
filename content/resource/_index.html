Here we’ve assembled all of the external resources for the workshop. These are templates that you can use to get started with different documents. Each item in the list below is linked to a private GitHub repo. Anyone who has attended this workshop should have access to these repos - if not, please let me know.

  • Example workflow in R: This is its own GitHub repo that assembles a hypothetical paper based on a built-in dataset within R. The main point of this example workflow is to ensure our research is easily reproducible (both to others and to our future selves). For context and extra description, see the accompanying videos:
    • Initiate project
    • Run project
    • Write papers
    • Correction In the previous video, I was “showing” the abstract and paper PDFs on a different screen (which was not being recorded). This video just shows those PDFs.
  • Example Academic Website: This is part of the ASHEcon newsletter series, where Alex Hollingsworth provides some great advice. If you’re having trouble with the link, just take a look at the example site here and the template repo here.
  • CV templates: A couple of examples from which you can get your own CV up and running
  • Cover letter templates: You can use these templates for a cover letter for your job market applications as well your journal submissions
